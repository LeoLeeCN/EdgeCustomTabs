# Edge Custom Tabs

Microsoft Edge is base on Google Chromium. Microsoft Edge custom Tabs add some extentsion functions for [Chromium Custom Tabs](https://github.com/GoogleChrome/custom-tabs-client/tree/master/customtabs).
